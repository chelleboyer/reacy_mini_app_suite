## HTML Entity Format

All emojis use decimal HTML entities in the format `&#[code];` or `&#[code];&#65039;` (with variation selector).

Example:
```html
<button>&#128077; Nod Yes</button>
```

This ensures emojis display correctly across:
- All browsers (Chrome, Firefox, Safari, Edge)
- All operating systems (Windows, macOS, Linux)
- All devices (Desktop, tablet, mobile)
